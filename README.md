# my-app

## 项目搭健过程
### 如何搭建vue3 + Quasar + electron项目
1、第一步创建vue3项目，选择自定义组件，选上router和vuex
```
vue create my-app
```
2、添加quasar
```
vue add quasar
```
3、添加electron即可
```
vue add electron-builder
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
